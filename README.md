# React + Redux Toolkit Task Manger App

# Features 
- Searching,Sorting
- User can perform create,read,update and delete operations
- Dark/Light Mode
- Toast Notifications
- Data Stored in local storage

# Tech Stack
- ReactJs
- Tailwind Css
- Redux Toolkit
- Context Api
- LocalStorage
- React Notifications
